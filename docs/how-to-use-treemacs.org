#+TITLE: Treemacs の使い方
#+DATE: <2026-01-18 Sun>

* 基本操作
| key  | 動作       |
|----+---------|
| RET / TAB | ファイルを開く、ディレクトリを展開・折りたたむ |
| o  | ファイルを他のウィンドウで開く   |
| ov  | 縦分割で開く      |
| oh  | 横分割で開く|

* ナヴィゲーション
| key  | 動作   |
|----+-----|
| n / C-n | 次の行   |
| p  C-p  | 前の行   |
| u  | 親ディレクトリへ移動 |
| M-n / M-p | 同階層の次・前の項目へ |
|H| ルートディレクトリへ |


* ファイル・ディレクトリ操作

| キー | 動作                                 |
|------+--------------------------------------|
| cf   | 新規ファイル作成                     |
| cd   | 新規ディレクトリ作成                 |
| R    | リネーム                             |
| d    | 削除                                 |
| m    | ファイルを移動                       |
| yr   | プロジェクトルートからの相対パスをコピー |
| ya   | 絶対パスをコピー                     |

* 表示・更新

| キー | 動作                     |
|------+--------------------------|
| gr   | 再読み込み               |
| th   | 隠しファイルの表示切替   |
| tw   | ツリー幅の変更モード     |
| q    | Treemacsウィンドウを閉じる |

* グローバルキーバインド (init.el設定)

| キー      | 動作                       |
|-----------+----------------------------|
| C-x t t   | Treemacsのトグル           |
| C-x t 1   | Treemacsウィンドウに移動   |
| C-x t d   | ディレクトリを選択して表示 |
