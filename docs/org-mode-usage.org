#+TITLE: Org-mode 使い方メモ
#+DATE: 2026-04-03

* 今日のタスクをピックする (SCHEDULED)

DEADLINE（締切）とは別に、「今日やる」タスクをマークするには =SCHEDULED= を使う。

** 操作

| キー      | 動作                         |
|-----------+------------------------------|
| =C-c C-s= | SCHEDULED を設定             |
| =C-c C-d= | DEADLINE を設定              |

日付入力画面で =.= を押すと今日の日付が入力される。

** 記法の例

#+begin_src org
** TODO タスク名
   SCHEDULED: <2026-04-03 Fri>
   DEADLINE: <2026-04-10 Fri>
#+end_src

** アジェンダでの確認

=C-c a a= (Day view) を開くと、今日 SCHEDULED されたタスクが一覧表示される。

* タスクの日付・ステータスを編集する

タスクのある行にカーソルを置いて操作する。

** キー一覧

| キー      | 動作                                        |
|-----------+---------------------------------------------|
| =C-c C-d= | DEADLINE を変更                             |
| =C-c C-s= | SCHEDULED を変更                            |
| =C-c C-t= | TODOステータスを変更 (TODO → DOING → DONE...) |

** 日付入力画面でのキー操作

| キー              | 動作           |
|-------------------+----------------|
| =.=               | 今日の日付     |
| =S-←= / =S-→=    | 1日ずつ移動    |
| =S-↑= / =S-↓=    | 1週間ずつ移動  |
| =Enter=           | 確定           |

既存の日付スタンプ =<2026-04-03>= の上でも =S-←= / =S-→= で直接増減できる。

* プロジェクト別タスク管理

** todo.org の構造

第1レベル見出しをプロジェクト名にする。

#+begin_src org
* ProjectA
** TODO タスクA1
** TODO タスクA2

* ProjectB
** TODO タスクB1
#+end_src

** Capture でのプロジェクト選択 (=C-c c t=)

=C-c c t= 実行時にプロジェクトを選択できる。

- 既存プロジェクトは補完候補として表示される
- 新規プロジェクト名を入力すると =todo.org= の末尾に自動作成される

#+begin_example
プロジェクト: [ProjectA / ProjectB / ...]  ← 補完で選択 or 新規入力
#+end_example

* DONE/CANCEL タスクを非表示にする

=C-c h= で DONE・CANCEL 済みタスクの表示・非表示をトグルできる。

| キー    | 動作                              |
|---------+-----------------------------------|
| =C-c h= | DONE/CANCEL タスクを非表示        |
| =C-c h= | もう一度押すと全タスクを再表示    |

実行後、ミニバッファに =DONE タスク: 非表示 / 表示= とメッセージが表示される。
